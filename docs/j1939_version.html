  • Check for open connection on TP.CM transmission
  • Change name of internal J1939_Tp_ts static elements

Version 4.10 - 2023.08

  • Add function J1939_TpSetAckSourceAddress() for ACK handling
  • Fix RQST response for data size > 8 bytes
  • Fix API call to J1939_MgrSetAddress()
  • Fix ECU address in template for J1939_MgrInit()
  • Fix example for ACK message

Version 4.08 - 2022.06

  • Add additional PGN handler result codes
  • Add function for destination address retrieval
  • Add example for PropA message
  • Fix ACK and RQST handling

Version 4.06 - 2022.03

  • Add CAM11 and CAM21 support
  • Add support for DM13
  • Add function to set PGN counter
  • Add example for STM32K476G-EVAL and STM32F769I-EVAL board

Version 4.04 - 2021.02

  • Add example for STM32091C-EVAL board
  • Add function to set PGN ticks
  • Improve static code checking
  • Fix parameter for logical network in examples

Version 4.02 - 2019.08

  • Update examples for several demo boards
  • Update documentation

Version 4.00 - 2019.04

  • Support multiple CAN channel
  • Support PGN list setup during run-time via J1939_PgnAddList()
  • New function J1939_PgnGetCycleTime()
